# WebPack 5 Boilerplate

## Packages

1. HtmlWebpackPlugin
2. CleanWebpackPlugin
3. MiniCssExtractPlugin
4. Autoprefixer
5. Css-loader
6. Cssnano
7. Postcss-loader
8. React
9. TypeScript

## Scripts

1. `npm run build` - bundle project
2. `npm run dev` - run DevServer
3. `npm run deploy` - deploy dist folder to GitHub Pages
